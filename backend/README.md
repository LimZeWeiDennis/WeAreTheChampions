# WE-ARE-THE-CHAMPIONS

**Prerequisites for system**

- Mongoose

## Backend setup

- Start a Terminal inside the "backend" folder
- Run `npm install` to set up the node dependencies
- Run `node index.js` to spin up local frontend at `http://localhost:5001/`
